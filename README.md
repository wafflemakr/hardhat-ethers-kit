# Hardhat Ethers Kit

Hardhat basic usage with ethers, waffle, forked mainnet and console.log

## Dependencies

Install dependencies with `npm install` or `yarn`

## Compiling

```
npm run compile
yarn compile
```

## Testing

```
npm run test
yarn test
```

## Coverage

```
npm run coverage
yarn coverage
```

## Deploying

```
npm run deploy
yarn deploy
```
